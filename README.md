# Gitlens - Github Profile Viewer

A modern web project focused on practicing API request handling, data fetching, and dynamic UI updates using JavaScript.

## Features

- Enter a GitHub username and fetch user data
- Display profile details including avatar, followers, repositories, and more
- Structured and clean UI for easy viewing
- Learn API handling and asynchronous JavaScript
